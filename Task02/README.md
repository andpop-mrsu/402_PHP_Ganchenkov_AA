# Task02: Progression Web App

Веб-приложение на PHP для игры "Арифметическая прогрессия" с хранением результатов в SQLite.

## Структура

- `public/` - корень сайта для встроенного PHP-сервера.
- `db/` - каталог для файла базы данных SQLite.
- `src/` - PHP-модули и шаблоны страниц.

## Запуск

1. Перейти в каталог проекта:

```bash
cd Task02
```

2. Запустить встроенный сервер PHP:

```bash
php -S localhost:3000 -t public
```

3. Открыть приложение в браузере:

```text
http://localhost:3000/
```

История игр и файл базы данных будут создаваться автоматически при первом запуске.
