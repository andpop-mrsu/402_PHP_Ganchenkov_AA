# Task01: Progression

Консольная игра на PHP, в которой нужно определить пропущенное число в арифметической прогрессии.

## Структура

- `Task01/progression/src` - модули приложения.
- `Task01/progression/bin` - исполняемый скрипт игры.
- `Task01/progression/composer.json` - конфигурация Composer.

## Запуск

После установки зависимостей:

```bash
cd Task01/progression
php bin/progression
```
